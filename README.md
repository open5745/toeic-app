# 多益行動學習助手（PWA 版 MVP）

聽、讀、單字一站式多益練習，搭配間隔重複（SRS）與每日提醒。
純 HTML / CSS / JavaScript，零框架、零建置工具、零依賴。

## 如何執行

Service Worker 與資料載入需要透過 HTTP 伺服器（不能直接雙擊 index.html）：

```
cd 專案資料夾
python -m http.server 8000
```

打開瀏覽器進入 <http://localhost:8000> 即可。
手機測試：手機與電腦連同一 Wi-Fi，用電腦的區網 IP 訪問（例如 http://192.168.x.x:8000）。

## 功能

| 模組 | 內容 |
|---|---|
| 單字 | 27 個情境單字包共 540 字（辦公室/人事/旅遊/採購/會議/行銷/財會/科技/餐飲/生活/物流/製造/不動產/醫療/環境/法務/媒體/教育/藝文/高頻動詞/形容詞/副詞/搭配語/客服/活動/銀行金融/交通運輸），單字卡含音標/詞性/發音/中英例句，簡化版 SM-2 間隔重複 |
| 文法 | 240 題 Part 5 & 6（含字彙選擇、假設語氣、倒裝、主詞一致等 800+ 分題型），優先出沒做過的題目，附中文解析與破題技巧 |
| 聽力 | 80 題 Part 1–4、美/英/加/澳口音（瀏覽器 TTS）、語速 0.8x–1.2x、單句重播、逐字稿開關、已完成標記與隨機出題 |
| 考試模式 | 隨機抽聽力 6 題（只播一次）+ 閱讀 12 題（限時 8 分鐘），交卷後統一錯題檢討；不受每日份量限制 |
| 每日份量 | 打開 App 只出當天的量：新單字 10、文法 10 題、聽力 3 題（複習卡另計）。全部完成後可按「繼續學習」追加半日份（+5/+5/+2），想加幾次都可以 |
| 首頁 | 連續學習天數（Streak）、最近 7 天學習量長條圖、今日任務進度 |
| 設定 | 每日提醒時間與自訂推播文案、清除學習紀錄 |

題量規劃：以每天 10 個新單字計，單字可撐 54 天；文法 10 題/天可撐 24 天、聽力 3 題/天可撐 26 天不重複。長期目標為單字 1000+、文法 300+、聽力 100+，資料結構支援隨時分批擴充。

## 專案結構

```
index.html            單頁應用外殼（底部分頁導覽）
manifest.webmanifest  PWA 安裝設定
sw.js                 Service Worker（離線快取、通知點擊）
css/style.css         行動優先樣式（支援深色模式）
js/
  app.js              路由與首頁（Streak + 圖表）
  storage.js          localStorage 包裝
  srs.js              間隔重複演算法（純函式）★
  streak.js           連續天數與活動統計（純函式）★
  speech.js           語音合成（口音/語速）
  vocab.js            單字卡 UI
  grammar.js          文法測驗 UI
  listening.js        聽力播放器 UI
  settings.js         提醒設定與排程檢查
data/
  vocab.json          單字包 ★
  grammar.json        文法題庫 ★
  listening.json      聽力題目與逐字稿 ★
icons/                App 圖示
```

★ 標記的檔案與 UI 完全解耦，日後轉 Flutter 可直接沿用（JSON 原樣搬走、演算法照邏輯翻成 Dart）。

## 已知限制（MVP 階段刻意取捨）

- **推播通知**：App / 瀏覽器開啟時準時提醒；完全關閉時要收到推播，需日後加上伺服器端 Web Push（或轉原生 App）。iOS 需將 PWA「加入主畫面」後通知才可用（iOS 16.4+）。
- **聽力語音**：使用瀏覽器內建 TTS，口音種類依裝置安裝的語音而定（Windows/Edge 的 Natural 語音效果最好）。日後可換成真人錄音檔，只需改 `speech.js`。
- **資料儲存**：進度存在 localStorage，清瀏覽器資料會遺失。日後可加匯出/雲端同步。
